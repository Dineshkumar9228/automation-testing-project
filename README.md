# 🚀 Automation Testing Project (Playwright)

![CI](https://github.com/Dineshkumar9228/automation-testing-project/actions/workflows/playwright.yml/badge.svg)

## 📌 Overview
This project demonstrates an end-to-end automation testing framework using Playwright with Page Object Model (POM). It covers UI workflows, API validation, and CI/CD integration.

## ⚙️ Features
- ✅ UI Automation (Login → Cart → Checkout)
- ✅ API Testing (User API validation)
- ✅ Page Object Model (POM)
- ✅ CI/CD using GitHub Actions
- ✅ HTML Test Reporting
- ✅ Screenshot & Trace on failure

## 🛠 Tech Stack
- Playwright (JavaScript)
- Node.js
- GitHub Actions (CI/CD)

## 🧪 Test Scenarios
1. User login with valid credentials  
2. Add product to cart  
3. Complete checkout process  
4. Validate order success message  
5. API response validation  

## 📂 Project Structure
tests/
pages/
utils/
.github/workflows/



## ▶️ How to Run Locally
npm install
npx playwright test


## 🔄 CI/CD Pipeline
- Tests run automatically on every push to `main`
- Workflow installs dependencies, runs tests, and uploads report

## 📊 Test Report
Playwright HTML report is generated and stored as an artifact in GitHub Actions.
